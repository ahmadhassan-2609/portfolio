# Video Directory

Place your video demonstrations here.

## Required File:
- `assistive-arm-demo.mp4` - Video demonstration of the Voice-Controlled Assistive Robotic Arm

## Recommended Specifications:
- **Format**: MP4 (H.264 codec) or WebM for better compression
- **Resolution**: 1920x1080 (1080p) or 1280x720 (720p)
- **Duration**: Any length (embedded player has controls)
- **File Size**: Try to keep under 50MB if possible (or optimize for web)

## Alternative Formats:
You can also provide:
- `assistive-arm-demo.webm` - WebM format for better browser support

## Video Optimization Tips:
1. **Compress**: Use tools like HandBrake or online compressors
2. **Export Settings**: 
   - Codec: H.264
   - Bitrate: 2-5 Mbps
   - Frame rate: 30 fps
3. **Trim**: Keep only essential parts to reduce file size

## Video Hosting Alternatives:
If the video file is too large, consider:
- Uploading to YouTube and embedding
- Using Vimeo
- Using cloud storage (Google Drive, Dropbox) with public links

## How It Works:
- Clicking the Assistive Robotic Arm project card will open a modal
- Video will auto-play when modal opens
- Modal can be closed by clicking X or pressing Escape
- Video controls (play/pause, volume, fullscreen) are available
