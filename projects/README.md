# Project Images Directory

Place your project images in this directory with the following names:

- `dg-slam.jpg` - Image for DG-SLAM project
- `ur5e-kinematics.jpg` - Image for UR5e Kinematics project
- `robotic-finger.jpg` - Image for Robotic Finger project
- `assistive-arm.jpg` - Image for Assistive Robotic Arm project

## Recommended Image Specifications:
- **Format**: JPG or PNG
- **Dimensions**: 1000x600px (16:9 aspect ratio recommended)
- **Size**: Optimized for web (under 500KB per image)
- **Content**: Screenshots, diagrams, or visual representations of your projects

## Adding Images:
1. Export or save your project images with the names above
2. Place them in this `projects/` directory
3. The portfolio will automatically display them

## Image Ideas:
- Screenshots of working projects
- Diagrams or visualizations
- 3D renders or CAD models
- Project demonstrations
- UI/interface screenshots
